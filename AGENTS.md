# AGENTS.md

## Package manager

- Use **pnpm**. Lockfile is `pnpm-lock.yaml`.

## Dev server

- `pnpm dev` runs Vite on **port 3000**.

## Code style

- Prettier: **no semicolons**, double quotes, trailing commas.
- TypeScript: `verbatimModuleSyntax` is enabled — use `import type` for type-only imports. `noUnusedLocals` and `noUnusedParameters` are enabled — unused variables will fail the build.
- Format and fix: `pnpm format`
- Check formatting: `pnpm check`
- Lint only: `pnpm lint`

## Routing

- Uses **TanStack Router** with file-based routing.
- Routes live in `src/routes/`.
- **Layout routes** are prefixed with `_` (e.g., `_main-layout.tsx`, `_authenticated.tsx`).
- `src/routes/__root.tsx` is the root route and wraps the app with providers (QueryClient, ThemeProvider, TooltipProvider).
- `src/routeTree.gen.ts` is **auto-generated** — do not edit it. The Vite plugin regenerates it on dev/build; you can also run `pnpm generate-routes` manually.

## Path aliases

- Prefer `#/*` over `@/*` for imports (e.g., `#/components/ui/button`). Both are mapped in `tsconfig.json`, but the codebase and `package.json` `imports` field use `#/*`.

## UI / Styling

- **Tailwind CSS v4** (no traditional `tailwind.config.js`). Theme tokens are defined in `src/styles.css` using `@theme inline`.
- Uses **shadcn/ui** (`radix-luma` style, non-RSC). Components are in `src/components/ui/`.
- The `shadcn` CLI is available if adding new components.

## Data fetching

- Uses **TanStack Query** (React Query).
- Pattern: `src/services/` holds raw fetch functions; `src/query-options/` exports `queryOptions` wrappers. See `src/query-options/hero-slides.ts` as the canonical example.
- **For the moment, all data returned by services must be hardcoded in-memory data** — there is no real backend yet.

## Tests

- Test runner is **Vitest** with jsdom, but **no tests exist yet**. Run with `pnpm test`.

## Build

- `pnpm build` produces output in `dist/`.
- `pnpm preview` serves the production build locally.

## Generated files to ignore

- `src/routeTree.gen.ts` — generated by TanStack Router. It is marked read-only in `.vscode/settings.json` and excluded from search/watch.
